# Twitter Sentiment Analysis

## Project Overview
This project uses Machine Learning and Natural Language Processing (NLP) to classify tweets as Positive or Negative.

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-Learn
- TF-IDF
- Naive Bayes

## Dataset
Twitter Training Dataset

## Features
- Text Cleaning
- Stopword Removal
- Stemming
- TF-IDF Vectorization
- Sentiment Prediction

## Accuracy
Accuracy = 0.8521446007206789
